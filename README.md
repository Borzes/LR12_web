# Лабораторная работа №12

Выполнил: Ащеулов Борис 8И41

## Описание проекта

SPA приложение состоящее из 3 страниц:

1. **Главная страница** - общая информация о проекте
2. **API страница** - получение и отображение данных с внешнего API
3. **Форма** - отправка данных через POST запрос

## Технологии

- Vue 3 (Composition API)
- Vue Router 4 (маршрутизация)
- Pinia (управление состоянием)
- Vite (сборщик)

## Установка и запуск

```bash
npm install
npm run dev
```
